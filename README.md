### About Me

I'm a Full Stack Web Developer with a BSCS from the University of South Asia, focused on building performant, visually distinctive web applications. My work spans front-end development, back-end systems, and WordPress-based platforms, with a growing focus on integrating AI/ML into practical, production-ready products.

**Core Focus Areas**
- Full stack web development — from responsive front-ends to server-side logic and deployment
- Applied AI/ML — predictive modeling and classification systems (e.g. GPA prediction, phishing URL detection)
- Interactive UI/UX — motion-driven interfaces using Three.js, GSAP, and modern animation libraries
- E-commerce and CMS platforms built on WordPress and custom stacks
